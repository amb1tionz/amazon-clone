# Amazon Clone
Made just for fun and educational purposes using [Fake Store API](https://fakestoreapi.com/)


### Built using:
- [x] ReactJS
- [x] Next.js
- [x] Redux Toolkit
- [x] Webhooks
- [x] NextAuth
- [x] Firestore
- [x] Stripe Checkout

## Instalation Guide
### Using npm
Run commands
1) ```npm install```
2) ```npm run dev```

## Or using yarn
Run commands 
1) ```npm install --global yarn```
2) ```yarn install```
3) ```yarn run dev```
